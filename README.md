# Color Blind
**Instant HEX / RGB / Name Color Previewer**

A minimalist, high-speed color inspection tool for developers and designers. No more opening heavy design software just to check a hex code.

## 1 Tool / 1 Action
- **Paste & See**: Paste any color code (HEX, RGB, or CSS Color Name) and the entire preview area instantly transforms.
- **Auto-Correction**: Missing a `#`? No problem. The tool automatically detects and corrects 3-digit and 6-digit hex codes.

## Features
- **Large Preview**: A generous preview area to truly feel the impact of the color.
- **Contrast Check**: Automatically determines if "Dark Text" or "Light Text" should be used based on W3C brightness standards.
- **Inverted Color**: Instantly calculates the inverted (complementary) color.
- **1-Tap Copy**: Tap the Inverted Color card to copy the HEX code to your clipboard immediately.

## Design Philosophy
- **Minimalist UI**: White background, #333 main text, and signature 15px rounded corners.
- **Performance**: Zero-latency, browser-only logic. Works perfectly on low-spec mobile and desktop.
- **Privacy First**: No login, no tracking, and no ads.

## How to Install
1. Open the tool in your browser.
2. **iPhone (Safari)**: Tap 'Share' -> 'Add to Home Screen'.
3. **Android (Chrome)**: Tap 'Menu (⋮)' -> 'Add to Home Screen'.
